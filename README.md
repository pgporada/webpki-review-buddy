# Usage

In a VM with Claude code, populate the working directory with WebPKI policy documents. Outputs a `CLAUDE.md` for use with Anthropic's LLM.
```
git clone https://github.com/pgporada/webpki-review-buddy
cd webpki-review-buddy
./setup-repos.sh
claude
```

Thanks to [Preston12321](https://github.com/Preston12321) for the initial `CLAUDE.md`.
